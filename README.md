Introduction
============
This project contains code that includes a monte carlo python evaluator. This is used to evaluate the following
integral:

x^2*y + e^(-z) dxdyzdz where the limits of integration are:

x: 0 to 2
y: -1 to 1
z: 1 to 125


How to Run
==========

`$ python montecarlo.py`
